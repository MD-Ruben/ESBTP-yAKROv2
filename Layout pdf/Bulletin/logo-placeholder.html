Instructions pour le logo

Ce fichier est un placeholder pour le logo de l'ESBTP. Pour utiliser un logo réel dans le bulletin :

  1. Créez ou obtenez une image du logo de l'école
  2. Renommez cette image en "logo.png"
  3. Placez-la dans le même dossier que les fichiers HTML, CSS et JS du bulletin

Le logo sera automatiquement affiché dans le bulletin.
